# AtCoder

僕のAtCoder管理用です。  
AtCoderアカウントは[こちら](https://atcoder.jp/users/kokastar)

## 各ディレクトリ
コンテストに提出したソースコードを入れています

## template
テンプレファイルを入れています

## AtCoder.cpp
AtCoderの問題を解くときメインで使ってるやつです。

## Atcoder.py
AtCoderの問題をPythonで解くときに使ってるファイルです

## input.txt
入力をリダイレクトするためのテキストファイルです。一回ごとに入力する手間が省けます。

# 使用言語
メイン:C++
サブ:Python
